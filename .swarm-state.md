## Still Open / Deferred

1. **Physical 5-slot support**: Orchestrator handles N=5 logically; renderer texture arrays are hardcoded to 3. Extending requires dynamic `pingPongTextures[]` allocation and bind-group changes.
2. **True parallel compositing**: Currently all parallel slots write to `writeTex`; only last survives. Needs per-slot output textures + blend pass.
3. **Depth texture scale mismatch**: `depthRead` is full-res, `depthWrite` is scaled. Should unify.
4. **87 generative shaders still ignore input**: WGSL-level fix, not renderer-level. Tracked in `LAYER_CHAIN_FIX_SUMMARY.md`.
