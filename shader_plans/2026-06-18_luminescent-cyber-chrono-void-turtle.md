# New Shader Plan: Luminescent Cyber-Chrono Void-Turtle

## Overview
A majestic, colossal cybernetic space-turtle slowly navigating a deep-space nebula, its intricate shell constructed of floating geometric chronal-plates that shift and emit liquid luminescent plasma in sync with cosmic frequencies.

## Features
- Intricate floating shell of interlocking geometric plates driven by domains and 3D noise.
- Bioluminescent "plasma veins" running between the plates, reacting to audio frequencies.
- Subsurface scattering effect on the plates to give them a cyber-organic look.
- Gravity wells generated by the mouse that slightly distort the space-time around the turtle.
- Slow, deliberate swimming motion powered by procedural domain warping and low-frequency oscillations.

## Technical Implementation
- File: public/shaders/gen-luminescent-cyber-chrono-void-turtle.wgsl
- Category: generative
- Tags: ["cyber-organic", "cosmic", "volumetric", "fractal", "animal"]
- Algorithm: Raymarching against complex combined SDFs involving a central spherical base and multiple voronoi/noise-distorted floating plates, with volumetric glow and chronal-distortion effects.

### Core Algorithm
- Primary domain uses a spherical SDF base for the body.
- The shell is formed by applying a 3D Voronoi-based displacement to create interlocking geometric plates.
- The gaps between plates use a thinner SDF that strongly emits light, driven by audio frequencies using `u.ripples`.
- Raymarching loop includes a soft-shadow pass and a volumetric glow accumulation pass.

### Mouse Interaction
- The mouse position (`u.config.xy`) creates a subtle gravity well.
- Rays passing near the gravity well are bent (spherical warp) altering the view of the turtle.
- Formula: `p = p + normalize(p - mouse_pos) * (strength / (length(p - mouse_pos) + 0.1))`

### Color Mapping / Shading
- Plates: Metallic, dark obsidian with subsurface scattering in the thin areas.
- Plasma Veins: Intense liquid neon glow using smoothstep over the Voronoi edges, mixed with deep cyan and bright magenta based on distance.
- Environment: A subtle volumetric starry nebula background using multi-octave FBM.

## Parameters (for UI sliders)
- Name: Shell Complexity (default: 0.5, min: 0.0, max: 1.0, step: 0.01) -> maps to u.zoom_params.x
- Name: Plasma Intensity (default: 0.8, min: 0.0, max: 2.0, step: 0.01) -> maps to u.zoom_params.y
- Name: Chrono-Distortion (default: 0.3, min: 0.0, max: 1.0, step: 0.01) -> maps to u.zoom_params.z
- Name: Swim Speed (default: 0.5, min: 0.1, max: 2.0, step: 0.01) -> maps to u.zoom_params.w
