# Showcase Shader 验收标准 v1.0

> 用于快速判断 Kimi Claw 生成的展示级着色器是否达到可合并标准。

---

## Idle 状态（未锁定）

- [ ] **12 秒吸引力**：在 12 秒自动轮播周期内，是否持续有视觉吸引力，不会让人觉得无聊或重复
- [ ] **运动流畅性**：动画是否自然流畅，没有明显卡顿、跳帧或低频闪烁
- [ ] **颜色与构图**：配色是否高级、有设计感，构图是否平衡且吸睛
- [ ] **催眠感**：是否具备 "hypnotic" 气质——让人想一直盯着看

## Mouse Claim 状态（锁定后）

- [ ] **视觉升级明显**：按 SPACE 后是否有明显、令人满意的视觉升级（强度/复杂度/速度增强）
- [ ] **交互自然**：鼠标交互是否自然且有反馈（拉扯、击碎、聚集等）
- [ ] **优雅回退**：松开鼠标或解锁后是否能优雅地回到 idle 状态，而不是突兀切换
- [ ] **Claim 时刻**：是否有明确的 "this is mine now" 的拥有感瞬间

## Audio Reactivity

- [ ] **频段对应合理**：Bass / Mid / Treble 是否有明显且合理的视觉对应（不是随机抖动）
- [ ] **增强美感**：音频反应是否增强了而不是破坏了原有视觉美感
- [ ] **跨类型稳定**：在不同音乐类型（低音重的电子乐、人声为主的流行、器乐等）下表现是否稳定
- [ ] **不过度反应**：安静时没有胡乱抖动，激烈时有明显反馈

## 技术与规范

- [ ] **naga 验证通过**：通过 `naga` 验证，无编译错误
- [ ] **规范符合**：符合 `WGSL_BUILTINS_GENERATIVE.md` 中的通用 uniform 结构
- [ ] **JSON 完整**：JSON 定义完整（包含参数说明 + showcase 标签）
- [ ] **性能良好**：无明显掉帧，目标 60fps on mid-tier GPU
- [ ] **移动端安全**：不使用 3D 纹理、过度嵌套循环、或未绑定的数组

## 评分标准

| 等级 | 条件 | 处理方式 |
|------|------|----------|
| **A+** | 所有检查项通过，有超出预期的亮点 | 直接合并，标记为 showcase 推荐 |
| **A** | 所有检查项通过 | 直接合并 |
| **B** | 1-2 个小问题 | 合并，记录 TODO 后续优化 |
| **C** | 3+ 个问题或核心功能缺失 | 退回，要求修复后重新提交 |
| **D** | 无法编译或严重视觉缺陷 | 直接丢弃，重新生成 |

---

> 使用方式：每个着色器生成后，用此清单逐项检查。5 分钟内完成验收。
