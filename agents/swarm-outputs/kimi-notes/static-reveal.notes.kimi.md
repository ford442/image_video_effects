# static-reveal — Kimi Upgrade Notes

## Changes
- Multi-layer static: fine noise + coarse noise with separate frequencies
- Per-channel chromatic displacement: R/B shift by treble amount
- Depth-reactive decay: foreground static clears faster
- Bass intensifies overall static flicker amplitude
- Mids drive noise pattern evolution speed

## Wow Factor
- Chromatic static looks like broken analog TV with color separation
- Depth layering makes static feel 3D

## Risks
- Two noise layers may be too busy on high-resolution sources
- Chromatic shift could cause moiré at certain noise scales
