# interactive-fresnel — Kimi Upgrade Notes

## Changes
- Gravitational lensing model: concentric Einstein rings
- Chromatic dispersion: R closer to center, B farther
- Depth acts as mass: deeper objects lens more
- Bass multiplies total lensing power
- Hue rotation matrix for rainbow ring coloring

## Wow Factor
- Rings warp space like a real gravitational lens
- Depth mass makes foreground objects feel heavy

## Risks
- Hue rotation matrix is verbose — could use lookup in future
- Ring aliasing at low `ringCount` values
