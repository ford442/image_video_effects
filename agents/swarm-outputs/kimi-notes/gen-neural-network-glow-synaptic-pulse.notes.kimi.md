# gen-neural-network-glow-synaptic-pulse — Kimi Notes

## Changes
- Chromatic synapse separation: excitatory pulses = warm amber, inhibitory = cool cyan, mixed by audio balance.
- Temporal potentiation: previous trail strengthens with repeated `bass` activation for long-term potentiation effect.
- Depth-scaled node density: `readDepthTexture` attenuates distant Voronoi nodes.
- Ripple interaction preserved and enhanced with chromatic pulse color.

## Wow-Factor
- Neural network that literally learns — repeated bass beats strengthen synaptic trails.
- Excitatory/inhibitory color coding makes the network topology readable at a glance.

## Risks
- 50 ripple loop is expensive; already present in original, unchanged.
- Potentiation can cause runaway brightness if decay is too low; `trail_decay` parameter controls it.
