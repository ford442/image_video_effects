# sonar-pulse — Kimi Batch C Notes

## What I Changed
- Added chromatic echo: R/G/B rings sample at slightly different distances for color separation.
- Added interference beats from a secondary detuned wave.
- Depth attenuates return signal (background objects are dimmer/harder to "ping").

## What I'm Proud Of
The chromatic echo makes each sonar ring look like it's refracting through water — the interference beats create moiré patterns that feel like real wave physics.

## What Might Need a Human Eye
- The detuned wave (1.03× freq) can cause flickering moiré at certain parameter combinations.
