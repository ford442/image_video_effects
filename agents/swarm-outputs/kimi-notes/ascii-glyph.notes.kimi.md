# ascii-glyph — Kimi Upgrade Notes

## Changes
- SDF glyph approximation: cross/dot shapes per cell
- Bass character swap: glyph pattern changes on beats
- Depth luminance map: foreground glyphs brighter
- Per-cell density based on luminance + densityBoost
- Treble adds chromatic highlights to dense characters

## Wow Factor
- Characters morph on every beat like a live terminal
- Depth tiering makes text feel like stacked layers

## Risks
- SDF approximation is basic — could use more glyph variety
- Small glyph sizes become unreadable noise
