# Ethereal Silk 设计文档（参考标准）

> 用于验收 Kimi Claw 生成的 Ethereal Silk 着色器

## 基本信息
- **着色器名称**：Ethereal Silk
- **视觉家族**：有机流动 / 丝绸质感
- **目标用途**：Generative Showcase 核心展示着色器

## 1. 整体视觉概念
- 像极轻的、半透明的发光丝绸/薄纱在空间中缓慢飘动。
- 带有柔和的光晕和微弱的内部结构（类似丝绸纤维或轻微褶皱）。
- 色调以冷白、淡金、浅紫、雾蓝为主，高级且梦幻。

## 2. Idle 状态（未锁定）核心要求
- **运动**：非常缓慢、优雅、有呼吸感。丝绸应该有自然的“飘”和“回弹”。
- **视觉吸引力**：12 秒内不能让人觉得无聊。要有持续但不突兀的光影变化和褶皱流动。
- **复杂度**：中低复杂度，强调质感和光感，而不是高频细节。
- **hypnotic 感**：需要让人愿意一直看下去，像在看缓慢飘动的窗帘或轻纱。

## 3. Mouse Claim 状态（锁定后）核心要求
- 鼠标应该像“手”一样对丝绸产生**拉扯 / 牵引 / 撕裂**的效果。
- 丝绸被鼠标吸引的部分应该明显变亮、变紧、流动速度加快。
- 松开鼠标后，丝绸应该有自然的回弹和重组过程（不要瞬间恢复）。
- Claim 后的变化要**明显且令人满足**，但不要破坏整体美感。

## 4. 音频反应映射建议

| 音频频段 | 映射目标 | 建议强度 | 说明 |
|----------|-------------------|----------|--------------------------|
| **Bass** | 整体流动速度 + 幅度 | 中 | 低频让丝绸“呼吸”更明显 |
| **Mid** | 褶皱波动 / 抖动 | 中高 | 中频驱动丝绸的细微波动 |
| **Treble** | 边缘高光闪烁 | 中 | 高频增加丝绸边缘的光感 |

## 5. 参数建议（zoomParam1-4）
- `zoomParam1`：流动速度 + 整体幅度
- `zoomParam2`：光晕 / 发光强度
- `zoomParam3`：褶皱复杂度 / 细节量
- `zoomParam4`：高光强度 + 边缘锐利度

## 6. 技术与展示要求
- 必须符合 `WGSL_BUILTINS_GENERATIVE.md`
- Idle 时性能要好，适合长时间轮播
- Claim 时可以适当增加计算量
- 颜色要高级，不能太“塑料”或廉价

## 7. 验收标准（对照 checklist）
- [ ] Idle 12 秒内持续吸引人，不重复
- [ ] 运动自然流畅，有呼吸感
- [ ] 颜色高级梦幻（冷白/淡金/浅紫/雾蓝）
- [ ] Claim 时拉扯/牵引效果明显且令人满足
- [ ] 松开后有自然回弹，非瞬间恢复
- [ ] Bass 控制整体流动，Mid 控制褶皱，Treble 控制边缘高光
- [ ] naga 验证通过，无编译错误
- [ ] JSON 配置完整，含 showcase 标签
- [ ] 性能良好，60fps 稳定
