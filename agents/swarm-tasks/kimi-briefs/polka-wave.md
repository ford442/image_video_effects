# KIMI BRIEF — polka-wave

## Creative Spark
A true CMYK halftone print engine: four dot grids at canonical print angles (15°, 45°, 75°, 90°) that collectively reproduce the input image. Mouse creates an expanding ripple that inverts dot polarity — dots become holes and holes become dots. Bass inflates all dots globally; treble adds micro-dot noise. The result should feel like a living Risograph print.

This week we are pushing: **print-native CMYK halftone + interactive ripple inversion + audio dot inflation**. Do not produce generic monochrome dots or simple circles.
